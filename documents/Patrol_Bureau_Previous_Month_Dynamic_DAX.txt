// ==========================================
// DYNAMIC PREVIOUS MONTH SUMMONS TOTALS
// ==========================================

// Total Parking Summons - Patrol Bureau - Previous Month (Dynamic)
Patrol_Bureau_Parking_Previous_Month = 
VAR CurrentDate = TODAY()
VAR PreviousMonth = EOMONTH(CurrentDate, -1)
VAR PreviousMonthStart = EOMONTH(CurrentDate, -2) + 1
VAR PreviousMonthEnd = EOMONTH(CurrentDate, -1)
VAR PreviousYear = YEAR(PreviousMonth)
VAR PreviousMonthNum = MONTH(PreviousMonth)
RETURN
CALCULATE(
    COUNTROWS('ATS_Court_Data'),
    'ATS_Court_Data'[WG2] = "Patrol Bureau",
    'ATS_Court_Data'[TYPE] = "P",
    'ATS_Court_Data'[Year] = PreviousYear,
    'ATS_Court_Data'[Month] = PreviousMonthNum
)

DYNAMIC BEHAVIOR:
- Today is September 8, 2025 → Shows August 2025 data
- Today is October 15, 2025 → Shows September 2025 data
- Today is January 3, 2026 → Shows December 2025 data

RECOMMENDED MEASURES TO USE:
1. Patrol_Bureau_Parking_Previous_Month_Date (most reliable)
2. Patrol_Bureau_Moving_Previous_Month_Date (most reliable)
3. Previous_Month_Label (for visual titles)

VISUAL SETUP EXAMPLE:
- Card 1: Patrol_Bureau_Parking_Previous_Month_Date
- Card 2: Patrol_Bureau_Moving_Previous_Month_Date
- Title: "Patrol Bureau Summons - " & [Previous_Month_Label]

HOW IT WORKS:
- EOMONTH(TODAY(), -1) gets the last day of previous month
- EOMONTH(TODAY(), -2) + 1 gets the first day of previous month
- Automatically handles year transitions (Dec → Jan)
- Automatically handles different month lengths
*/
